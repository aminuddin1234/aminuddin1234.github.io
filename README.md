## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ and npm

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/aminuddin1234/amin-portfolio.git
   cd amin-portfolio
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Start the development server
   ```bash
   npm run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser

### Building for Production

```bash
npm run build
```

### Deploy to GitHub Pages

```bash
npm run deploy
```

- **S**ituation - Context and problem statement
- **T**ask - Specific objectives and requirements
- **A**nalysis - Approach, tools, and methodology used
- **R**esult - Measurable outcomes and impact
